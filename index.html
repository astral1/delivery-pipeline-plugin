Delivery-Pipeline-Plugin

Plugin for Jenkins for pipeline visualisation, perfect for Continuous Delivery

Download this project as a .zip file Download this project as a tar.gz file

What is it?

Plugin for Jenkins for visualisation of pipelines made up of build jobs in Jenkins with upstream/downstream dependencies.

How to use it?

First of all you need to build it and install it. The plugin is not yet in the Jenkins plugin repository yet.


$ git checkout https://github.com/Diabol/delivery-pipeline-plugin.git
$ cd delivery-pipeline-plugin
$ mvn clean package
